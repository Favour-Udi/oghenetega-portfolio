# Oghenetega Udi — Developer Portfolio

A lightweight, GitHub-style personal portfolio built with vanilla **HTML/CSS/JS**. Optimized for GitHub Pages or any static host.

## Quick start

1. Replace your contact email in `index.html` (search for `your.email@example.com`).
2. Update `projects.json` with more projects.
3. (Optional) Add a headshot to `assets/profile.jpg` and link it in the header.
4. Push the folder to a new GitHub repo and enable **Settings → Pages → Deploy from Branch**.

## Structure

```
oghenetega-portfolio/
├─ index.html
├─ styles.css
├─ script.js
├─ projects.json
└─ assets/
```

## Customise
- Colors and dark mode: tweak CSS variables in `styles.css`.
- Sections: add/remove sections inside `index.html`.
